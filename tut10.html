Id's and Classes Tutorial

Important:
  1. One id(identifier) is applicable only once in a page
  2. Class is used for applying a property to multiple elements, using Css or Javascript that will be taught earlier.
  3. Class is a One Word thing, and multiple classes can be written for a single tag using spaces. For Eg:
    • Single Class: class="redBg"
    • Multiple Classes: class="regBg blackBorder Sanju"
    Here in the second example, the element will get linked to 3 different classes and all properties of that classes will get applied to this tag.
